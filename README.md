# 🛡️ Anonimizador de Documentos

Script interactivo en Python para anonimizar documentos comunes (`.pdf`, `.docx`, `.xlsx`, `.csv`, `.odt`). Sustituye automáticamente datos personales y sensibles como:

- 📧 Correos electrónicos  
- 🆔 DNI y CIF  
- 📞 Teléfonos  
- 🏢 Direcciones  
- 📁 Códigos y expedientes administrativos  
- 📆 Fechas  
- 🔑 Palabras personalizadas desde `anonimizables.txt`

---

## ⚙️ Requisitos

Necesitas tener instalado:

- Python 3.12  
- Tkinter (para la interfaz de selección)

Instala los paquetes necesarios con:

```bash
pip install -r requirements.txt
