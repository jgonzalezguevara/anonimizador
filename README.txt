# Anonimizador de Documentos

Este script permite anonimizar automáticamente archivos \`.docx\`, \`.xlsx\`, \`.pdf\`, \`.csv\`, y \`.odt\`, eliminando datos personales sensibles como DNIs, emails, teléfonos y otras palabras clave personalizadas.

## Uso

\`\`\`bash
python3 anonimizar.py
\`\`\`

Al ejecutar, aparecerá un selector para elegir el archivo y la carpeta de destino.

## Requisitos

Instala las dependencias con:

\`\`\`bash
pip install -r requirements.txt
\`\`\`
